# 🤖 AI & ML Practical Exercises

A structured, week-by-week collection of hands-on Jupyter Notebooks covering the full spectrum of Artificial Intelligence and Machine Learning — from Python fundamentals all the way to Diffusion Models and YOLO.

---

## 📚 Course Outline

| Week | Notebook | Topics Covered |
|------|----------|----------------|
| Week 1 | `week1_python_fundamentals.ipynb` | Python basics, data types, control flow, functions |
| Week 2 | `week2_numpy_visualization_OOP.ipynb` | NumPy arrays, Matplotlib/Seaborn visualization, OOP |
| Week 3 | `Week3_Regression_on_real_data.ipynb` | Linear & polynomial regression on real datasets |
| Week 4 | `week4_perceptron_logistic_DeepNNs.ipynb` | Perceptron, Logistic Regression, Deep Neural Networks |
| Week 5 – Part 1 | `Week5_Part1_Data_Preprocessing.ipynb` | Handling missing values, encoding, feature scaling |
| Week 5 – Part 2 | `Week5_Part2_Visualization_Multicollinearity.ipynb` | Correlation heatmaps, VIF, multicollinearity detection |
| Week 5 – Part 3 | `Week5_Part3_CrossValidation.ipynb` | K-Fold CV, train/val/test splits, model evaluation |
| Week 5 – Part 4 | `Week5_Part4_Advanced_Topics.ipynb` | Regularization, hyperparameter tuning, pipelines |
| Week 7 | `week7_probabilistic_ml.ipynb` | Naive Bayes, Gaussian Processes, Bayesian inference |
| Week 8 | `week8_sequence_models.ipynb` | RNNs, LSTMs, GRUs, time-series modeling |
| Week 9 | `week9_attention_transformers.ipynb` | Attention mechanism, Transformer architecture |
| Week 10 | `Week10_Diffusion_Models.ipynb` | Score matching, DDPM, generative diffusion models |
| Week 11 | `week11_CNNs_transfer_YOLO.ipynb` | CNNs, Transfer Learning, YOLO object detection |

---

## 🛠️ Tech Stack

- **Language:** Python 3.x
- **Environment:** Jupyter Notebook / Google Colab
- **Core Libraries:**
  - `numpy`, `pandas` — data manipulation
  - `matplotlib`, `seaborn` — visualization
  - `scikit-learn` — classical ML
  - `tensorflow` / `keras` or `pytorch` — deep learning
  - `transformers` (Hugging Face) — attention & transformers

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/fares-ahmed7/AI-ML-Practical-Exercises.git
cd AI-ML-Practical-Exercises
```

### 2. Install dependencies

```bash
pip install numpy pandas matplotlib seaborn scikit-learn tensorflow torch transformers
```

### 3. Launch Jupyter

```bash
jupyter notebook
```

Or open any notebook directly in [Google Colab](https://colab.research.google.com/) by clicking the badge in each notebook.
